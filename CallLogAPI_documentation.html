get /log
returns call log (getCallLog)
You can get call log from the host android device. It returns info on all the phone calls since the start of the server

Return type

array[CallLogEntry]

Example data

Content-Type: application/json
[ {
  "duration" : 123,
  "number" : "+34124783013",
  "beginning" : "2000-01-23T04:56:07.000+00:00",
  "name" : "John Doe",
  "timesQueried" : 4
}, {
  "duration" : 123,
  "number" : "+34124783013",
  "beginning" : "2000-01-23T04:56:07.000+00:00",
  "name" : "John Doe",
  "timesQueried" : 4
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

data returned correctly
Up 
get /status
returns current call status (getCallStatus)
Returns information whether there's an ongoing call on the host device and relevant information about the caller

Return type

CallStatus

Example data

Content-Type: application/json
{
  "number" : "+34123873821",
  "ongoing" : true,
  "name" : "John Doe"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

call status returned CallStatus
